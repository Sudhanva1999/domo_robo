Time-of-Flight sensor (VL53L0X / VL53L1X breakout)
ToF Pin    Pi Zero Pin         Notes
VIN      → Pin 1  [3.3V]      3.3V (breakout has onboard reg; also accepts 5V via Pin2)
GND      → Pin 9  [GND]
SDA      → Pin 3  [GPIO2]     I2C1 data  — enable I2C in raspi-config
SCL      → Pin 5  [GPIO3]     I2C1 clock — enable I2C in raspi-config
INT      → Pin 7  [GPIO4]     interrupt output (active LOW); optional
SHUT     → Pin 29 [GPIO5]     XSHUT / hardware reset (pull HIGH to run); optional
